ВНИМАНИЕ! По умолчанию версия шаблона для DLE 11.0.

1) Установка шаблона:
---------------------------------------------------------
- Закачиваем папку с шаблоном на сайт в папку templates
- Устанавливаем для закачанной папки с шаблоном права на запись (CHMOD 777), а для файлов внутри неё 666
- Открываем файл contacts.php и заменяем вверху email адрес на свой и закидываем этот файл в корень сайта (где лежит файл admin.php)
- Выбираем в настройках движка наш шаблон
- Отредактируйте логотип в файле main.tpl
- Отредактируйте файл include/main-content.tpl - он отвечает за вывод основного контента на главной странице
- В админпанели количество выводимых похожих новостей укажите 3 штуки. Чтобы изменения вступили в силу почистите кеш похожих новостей (перейти в раздел site.ru/admin.php?mod=rebuild  и в самом низу нажать Начать перестроение)
- Для новостей и комментариев установите рейтинг "Нравится или не нравится", либо "Только нравится"

2) Краткие новости в шаблоне
---------------------------------------------------------
shortstory.tpl - краткая новость по умолчанию в 3 колонки
shortstory2.tpl - краткая новость в одну колонку (на всю ширину)
shortstory3.tpl - краткая новость для раздела портфолио, или для демонстрации своих работ
Устанавливать данные шаблоны нужно в настройках ваших категорий в Шаблон краткой новости

3) Как изменить цвет шаблона
---------------------------------------------------------
Из папки "другие расцветки шаблона" копируем нужный цвет шаблона - edit.css и закидываем его в папку css в шаблоне, согласившись на замену.
Чтобы активировать boxed версию шаблона или переместить правую колонку на левую сторону, читаем что написано в комментариях в edit.css

4) Тег {custom}  или как настроить шаблон:
---------------------------------------------------------
Вывод новостей на главной реализован через специальный тег {custom}
Пример:
{custom category="3" template="shortstory-2" aviable="main" from="0" limit="6" cache="no"}
где:
category="3" - это id категории, из которой будут выводиться новости (можно перечислять несколько через запятую - нужный id можно посмотреть в админке напротив категории)
Находим в main.tpl нужный блок и проставляем id категории. Нужно указывать id той категории, в которой есть новости. Если нужно сделать вывод новостей из родительской категории, то нужно перечислить через запятую все подкатегории.

6) Настроим слайдер изображений в полной новости
---------------------------------------------------------
Создаем дополнительное поле:
Название поля: images
Описание поля: Изображения
Тип поля: Несколько строк
Значение по умолчанию: ничего не пишем
Использовать при желании - поставить галочку
Подключить редактор - поставить галочку

Чтобы выводить описание к каждой фотографии сделаем небольшой хак (если описание выводить вам не нужно, то данный хак не нужно делать):
Открыть файл /engine/classes/parse.class.php

Найти в 4-х местах:
$alt = "alt=\"" . $alt . "\" title=\"" . $alt . "\" ";

Заменить на:
$alt = "alt=\"" . $alt . "\" title=\"" . $alt . "\" data-caption=\"" . $alt . "\"";

При загрузке изображений для слайдера уменьшенные копии НЕ создаем (это важно), в итоге примерный вид вставленной в поле картинки будет такой:
[img]тут будет ссылка на картинку[/img]

Чтобы добавить описание к картинке, делаем так:
[img=|Описание для картинки 1]тут будет ссылка на картинку 1[/img]
[img=|Описание для картинки 2]тут будет ссылка на картинку 2[/img]
и так далее....

7) Настроика иконок в шаблоне
---------------------------------------------------------
Все иконки в шаблоне выводятся через шрифтовые иконки fonts awesome
Список всех иконок доступен на этой странице http://fontawesome.io/icons/
Чтобы заменять иконки находим в нужных местах их кода, например, <i class="fa fa-search"></i>
Тут нас интересует класс fa-search. По ссылке выше находим нужную иконку, например, beer
В итоге у нас получится <i class="fa fa-beer"></i>
И точно также с со всеми иконками, которые вам захочется поменять.
